Researching ways to expedite data analysis with Computer Vision. Currently, I am working on accurately identifying charcoal contours from image samples, which are no bigger than 5mm across, such that the script can accurately estimate the number of charcoal pieces in the entire sample, numbering in the thousands, which would significantly reduce the time and error compared to a manual count. I am using the OpenCV library for its Computer Vision modules to help with this.
